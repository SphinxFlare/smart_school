# 🏫 Smart School Management System

A modular, event-driven, domain-structured school governance platform designed for scalability, clarity, and long-term maintainability.

---

# 🚍 Driver Role

### Transport Operations
- Start / End transport session
- Student pickup confirmation
- Student drop confirmation
- Live location streaming
- ETA support (system-calculated)
- Delay reason entry (manual)

### Driving Behavior Monitoring
- Speeding detection
- Harsh braking detection
- Aggressive driving detection

# 👩‍🏫 Teacher Role

### Attendance Management
- Mark Present / Absent / Late
- Add absence reason (optional)

### Academic Control
- Create exam schedules:
  - Weekly
  - Monthly
  - Quarterly
- Confirm exam completion
- Enter marks

### Homework
- Assign homework
- Track homework completion

### Student Observation
- Log micro-observations
- Flag concerns:
  - Studies
  - Behaviour
  - Attention
  - Health
- Add short note (optional)

### Parent Communication
- One-way or two-way messaging
- Call redirection feature
- Restricted to attendance, academics, and concerns

### Class Overview
- Today’s attendance
- Upcoming exams
- Subject details
- Students with active concerns

# 📌 System Design Principles

- Domain-driven modular architecture
- Role-based access control (RBAC)
- Event-driven internal communication
- Async background processing
- Strict audit tracking
- Scalable foundation for multi-school expansion
